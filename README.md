# Sales Analytics SQL Project

## Project Overview
This project analyzes sales data using MySQL. The database contains Customers, Products, and Orders tables.

## Database Schema

### Customers
- customer_id
- customer_name
- city

### Products
- product_id
- product_name
- category
- price

### Orders
- order_id
- customer_id
- product_id
- quantity
- order_date

## SQL Concepts Used
- CREATE DATABASE
- CREATE TABLE
- PRIMARY KEY
- FOREIGN KEY
- INSERT
- SELECT
- JOIN
- GROUP BY
- ORDER BY
- Aggregate Functions
- Views
- Window Functions

## Business Questions Solved
1. Total Revenue Analysis
2. Top Customers Analysis
3. Best Selling Products
4. Monthly Revenue Analysis
